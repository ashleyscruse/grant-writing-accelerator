# Grant Writing Accelerator

Curriculum and facilitation materials for the **Grant Writing** track of the NAIRR Accelerator at the Morehouse Supercomputing Facility. A structured program that takes faculty from "I should apply for grants" to a working proposal draft and a repeatable, AI-accelerated workflow they keep using.

> It is not the AI. It is the context. Set up who you are once, and the same five-step workflow gives you grounded, honest help on every proposal.

## The two participant-facing artifacts

1. **The workbook** ([`docs/`](docs/)) — a GitHub Pages site organized by the five-stage framework. The keep-after resource: every step of the program is written up so participants can repeat the workflow after the week ends.
2. **The Grant Writing Starter** (separate repo: [`grant-writing-starter`](../grant-writing-starter)) — the context workspace participants fork. It wires their professional profile and two AI skills (grant-analyzer, grant-writing) into one folder, so AI tools can actually help.

## The framework

Five stages. AI accelerates each one; the researcher decides between stages. See [`framework.md`](framework.md).

| Stage | Focus | Deliverable |
|---|---|---|
| 1. Foundation | Build the profile that grounds every evaluation and draft | A reusable profile |
| 2. Opportunity | Find opportunities and evaluate fit honestly | A fit evaluation and a go decision |
| 3. Strategy | Define the project concept, aims, and team | Aims and a team document |
| 4. Drafting | Write sections mapped to the funder's rubric | Drafted proposal sections |
| 5. Review | Read as a reviewer, check compliance, submit | A review-ready, compliant draft |

## Structure

```
  framework.md            -- the five-stage AI-accelerated grant writing framework
  getting-started.md      -- participant onboarding (profile + skills setup)
  docs/                   -- GitHub Pages workbook (Jekyll), the keep-after resource
  templates/              -- profile and proposal templates (identity, research, team, etc.)

    tapis-demo-run-of-show.md   -- Monday, ~20 min, split-screen Tapis/TACC demo
    ai-demo-run-of-show.md      -- Thursday, live AI-for-grant-writing workflow demo
```

The participant workspace lives in the sibling [`grant-writing-starter`](../grant-writing-starter) repo.
